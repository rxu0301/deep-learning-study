# AI Learning Repository (2024-2026)

## 📚 개요

이 저장소는 초급 수준의 기초 개념부터 고급 딥러닝 프로젝트까지 2년간의 학습 여정을 담고 있습니다. 
각 폴더는 학습 시기와 학습 주제별로 구분되어 있으며, 실습 예제와 프로젝트를 통해 실제 데이터 분석과 머신러닝의 전체 프로세스를 경험했습니다.

- **학습 기간**: 2024년 9월 ~ 11월
- **핵심 주제**: 기초 Python 라이브러리, 기본 딥러닝 개념 입문
- **기술 스택**: NumPy, Pandas, Matplotlib, Seaborn, OpenCV, TensorFlow/Keras

  - OpenCV: 이미지 처리 (흑백 변환, 색상 처리)
  - 기본 딥러닝: Dense 신경망, 손글씨 인식, 시험 점수 예측, 이진 분류 소개
- **주요 프로젝트**: 손글씨 분류, 신경망 기초 학습, 이미지 처리 입문

- **기술 스택**: NumPy, Pandas, Matplotlib, scikit-learn, Seaborn
