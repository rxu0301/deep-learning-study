# Colab Notebooks - 머신러닝 실전 프로젝트 (2025년 하반기)

> 2025년 8월 ~ 12월, 머신러닝 알고리즘을 실제 데이터에 적용한 종합 실습 저장소입니다.

## 📚 학습 개요

이 저장소는 이전에 학습한 머신러닝 알고리즘들을 실제 데이터셋에 적용하고, 다양한 분석 문제를 해결하는 실전 프로젝트들을 담고 있습니다.
Google Colab 환경에서 진행된 학습으로, 복잡한 데이터 분석과 여러 알고리즘의 비교를 통해 실무 능력을 키웠습니다.

## 🎯 학습 내용 요약

### 1. 머신러닝 알고리즘 복습 및 심화
- 선형 회귀 (Linear Regression) 심화
- 로지스틱 회귀 (Logistic Regression) 실습
- K-최근접이웃 (KNN) 응용
- 나이브 베이즈 (Naive Bayes) 분류
- 서포트 벡터 머신 (SVM) 심화
- 랜덤 포레스트 (Random Forest) 응용
- K-Means 군집화 (Clustering)
- 주성분 분석 (PCA) 활용

### 2. 실전 데이터 분석
- 인기도 예측 (Popularity Prediction)
- 코로나바이러스 감염 현황 분석
- 대규모 데이터셋 처리
- 복합 특성 엔지니어링

### 3. 고급 분석 기법
- 모델 앙상블 (Ensemble)
- 하이퍼파라미터 튜닝 최적화
- 교차 검증 (Cross Validation)
- 모델 성능 비교 분석
