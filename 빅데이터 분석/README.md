### 머신러닝 알고리즘

#### 1. 선형 회귀 (Linear Regression)
- 개념: 선형 관계 학습을 통한 연속값 예측
- 적용: 집값 예측, 판매량 예측 등
- 성능: RMSE로 평가

#### 2. 로지스틱 회귀 (Logistic Regression)
- 개념: 확률 기반 분류
- 적용: 이진 분류, 질병 진단 등
- 성능: Accuracy, Precision, Recall

#### 3. 최근접이웃 (K-Nearest Neighbors)
- 개념: 가장 유사한 k개 샘플로 예측
- 적용: 추천 시스템, 유사도 분석
- 특징: 거리 계산 중요 (정규화 필수)

#### 4. 나이브 베이즈 (Naive Bayes)
- 개념: 베이즈 정리 기반 확률 분류
- 적용: 스팸 필터, 텍스트 분류
- 장점: 빠른 학습 및 예측

#### 5. 서포트 벡터 머신 (SVM)
- 개념: 마진 최대화를 통한 분류
- 적용: 복잡한 비선형 분류
- 커널: Linear, RBF, Polynomial

#### 6. 랜덤 포레스트 (Random Forest)
- 개념: 여러 결정 트리의 앙상블
- 적용: 특성 중요도 분석, 분류/회귀
- 장점: 과적합 방지, 해석 용이

#### 7. K-Means 군집화 (Clustering)
- 개념: 비지도 학습으로 k개 그룹 생성
- 적용: 고객 세분화, 이미지 압축
- 도전: 최적 k값 결정

#### 8. 주성분 분석 (PCA)
- 개념: 고차원 데이터를 저차원으로 축소
- 적용: 시각화, 특징 추출
- 유점: 분산 설명도로 평가
